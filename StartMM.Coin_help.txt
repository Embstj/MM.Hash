StatsInvterval = 1 for benchmarking, 1000 for benchmarking disable
Currency = Fiat Currency Of Your Choice
CoinExchange = Coin Symbol Of Your Wallet (Will only work for coins on cryptocompare)
Interval = Time before it starts to build database. Ideal target goal should be miner windows close/open every 5 minutes.
Location = Country
Poolname = Currently only zergpool supported
Algorithm = Algorithms you wish to mine
Proxy = Proxy Server Address (Not Tested)
Type = Seperate Device Groups. Up to 3 + CPU. The options are "CPU,NVIDIA1,NVIDIA2,NVIDIA3"
GPUDevices1 = Devices you are using for group. Example: '0,1,2,3'.  This must match Type "NIVIDIA1".  Leave '' if using CPU
Wallet1 = Wallet for device group 1
PasswordCurrency1 = Wallet Symbol (See Pool For More Details, Must Match Wallet)
GPUDevices2 = Devices you are using for group 2.
Wallet2 = Wallet for device group 2
PasswordCurrency2 = Wallet Symbol
GPUDevices3 = Same as above
Wallet3 = Same as above
GPUDevices3 = Same as above
Donate = percentage you wish to donate daily. Miner remembers when you open or close, so it will not steal from you...However it only resets donation switch if miner hasn't ran for 24 hours.
